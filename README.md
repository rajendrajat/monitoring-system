# Market Monitoring System

A Python-based monitoring system that tracks crypto prices, market indices, and news, sending alerts via Telegram.

## Features
- 📊 Real-time crypto price monitoring (Bitcoin, Ethereum)
- 📈 Market data tracking (NIFTY, BSE)
- 📰 News monitoring with keyword alerts
- 📱 Telegram bot notifications

## Setup
1. Clone the repository
2. Install dependencies: `pip install requests`
3. Run: `python main.py`
